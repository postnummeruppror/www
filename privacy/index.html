Integritetsskydd (Privacy policy)

Informationen vi samlar in används för att skapa en databas över postnummer. I samband med att du skapar ett konto registrerar vi ditt namn, din e-postadress och ett unikt ID för din enhet. Denna information delas inte med någon annan och vi använder den bara för att kontakta dig om det gäller något viktigt inom ramen för Postnummerupproret.

Avsluta konto

Om du önskar att vi tar bort din kontoinformation, kontakta oss via e-post på postnummeruppror@gmail.com.
