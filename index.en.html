What we want

Post codes, addresses and related data is used in many applications but is expensive to ge access to and maintain. In Sweden it will cost you aroound 5000 EUR per year to keep the data updated. This cost prevents a lot of people from using the data to build services.

Denmark studied the potential benefits of open address and post code data and discovered that the previous business model was outdated. We have reason to believe that this applies for Sweden too.

We want to create a new post code and address database free to use for everyone (CC0). At the same time we want to show decision makers that the current business model for swedish post code data is outdated.

As the original source for post code data is protected we need to create a new one from scratch. We need your help. By reporting address information in one of our applications you contribute to the database.

Who are you?

The post code revolution was created by Peter Krantz (@peterkz_swe) and Karl Wettin (@kodapan). Our code is available on github.

You can help

If you are in Sweden you can contribute by reporting addresses in your vicinity, e.g. where you live or when you have a receipt from a store you are at. Navigate to the web form and register the information.

We also have an app for Android.

Do you have a dataset containing address or post code data that you can share?

Would you like to help us develop collection apps for other platforms?
